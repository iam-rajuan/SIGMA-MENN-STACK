# NodeJS

- **Initialize npm**
```sh
npm init
```
- **Initialize without identifiy**
```sh
npm init -y
```
# ExpressJS

- **Initialize npm**
```sh
npm init
```
- **Install express**
```sh
npm i express
```
# Nodemond Reload server

- **Install Nodemon**
```sh
npm i nodemon
```
- **Install Nodemon Globally**
```sh
npm i --global nodemon
```
- **Start Nodemon**
```sh
nodemon file.js
```
